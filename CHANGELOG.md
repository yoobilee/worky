## [v1.1.0] - 2026-06-11

### ✨ 거래처 관리 개선
- 목록형 진행현황 패널 슬라이드 인/아웃 + 잔디밭 클릭 토글 기능
- 상태 드롭다운 fixed 포지션 (표 바깥으로 표시)
- 편집 모드 추가 (수정/삭제 체크박스 일괄 처리)
- 검색창 추가 (거래처명·담당자·태그·메모 등 통합 검색)
- 정렬 드롭다운 8종 (상태순·만료 임박·계약 시작일·거래처명·담당자 오름/내림차순)
- 거래처명 말줄임 + 커스텀 툴팁
- 태그 +N 호버 툴팁 (옆쪽 방향)
- 홀짝 행 색상 진행현황 패널과 동기화
- 다크모드 시인성 개선 (태그·잔디밭·툴팁)
- 도움말 업데이트

### ✨ Q&A 개선
- 대화 히스토리 자동 저장 (Supabase) + 히스토리 패널
- 히스토리 패널 슬라이드 애니메이션
- 추천 질문 랜덤화 (20개 풀에서 3개 추출)
- 추천 질문 UI 개선 (배경색·간격)
- 도움말 업데이트
