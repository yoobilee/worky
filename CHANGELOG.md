## [v1.4.0] - 2026-07-04

### 🔒 보안 & 성능
- /api/groq, /api/kakao-places API 라우트에 로그인 인증 체크 추가
- 캘린더/데이터 정리 히스토리 조회에 안전 상한(limit) 적용, 월별 조회로 전환
- Supabase daily-notifications Edge Function 자동 실행 복구 (pg_net 확장 활성화)
- 앱 레벨 에러/404 페이지 신설

### ♿ 접근성 개선
- 에러 메시지 전반에 role="alert" 적용
- 아이콘 전용 버튼에 aria-label 추가
- 탭 UI에 role="tab"/aria-selected 적용
- 색상 대비 전면 개선 (연한 회색·보라 텍스트 명도 조정)

### 🌐 다국어 지원 (한국어/영어)
- 설정, 거래처 관리, 구성원 관리, 캘린더, 사이드바, 홈 화면, 알림, AI 기능 10종 전체 다국어 전환
- 설정 페이지에 언어 설정 섹션 추가

### 🎨 홈 화면 & 디자인 전면 리뉴얼
- 홈 화면을 AI 요약 카드 + 핵심 지표 3종 + 실사용 기반 추천 기능으로 재구성
- 카드 호버, 결과 등장, 탭 전환 등 마이크로 애니메이션 전역 적용
- AI 기능 10종 UI를 라벨 중심 폼에서 pill 칩 + 통합 카드 형태로 리디자인
- Q&A에 대화 시작 전 추천 질문 랜딩 화면 추가, AI 답변 마크다운 렌더링 지원
- 설정 페이지를 아코디언에서 좌우 분할(목록+상세) 레이아웃으로 개편
- 모바일 반응형 개선 (홈 화면, 거래처 관리, 할 일/메모 등 레이아웃 수정)

### 🐛 버그 수정
- 카카오맵 장소 링크 오작동 수정, 장소 검색 결과 호버 툴팁 추가
- AI 결과 마크다운(**굵게**) 렌더링 누락 수정 (문서 요약 등)
- PDF 텍스트 추출 워커를 CDN 의존에서 로컬 번들로 전환
- Q&A 히스토리 중복 저장 버그 수정 (스트리밍 완료 시점에만 1회 저장)
- 이메일 답장 복사 아이콘 깨짐 수정, 답장 초안 3개 미생성 문제 개선
- 사용 통계 오분류 수정 (피드백 정리 기능이 Q&A로 잘못 집계되던 문제)
- 모달/패널이 화면 전체를 못 덮던 오래된 레이아웃 버그 수정 (포털 방식 적용)
